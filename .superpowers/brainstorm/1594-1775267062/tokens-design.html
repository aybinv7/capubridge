Typography & Spacing

Primary text — 14px Geist (labels, headings, nav items)
Secondary text — 12px Geist (descriptions, metadata, timestamps)
Tertiary text — 11px Geist (status bar, captions, counts)
Mono primary — 13px Geist Mono (code, values, ports)
Mono secondary — 11px Geist Mono (timestamps, IDs, hashes)

Key Design Rules

Radius
Panels/containers: 0px
Buttons/inputs: 4px (0.25rem)
Badges/pills: 4px
Shadows
All shadows: 0 opacity (flat)
Separation: 1px borders only
No blur, no glow effects
Transitions
Hover: color/bg 150ms ease
Panels/sheets: translateY 200ms
Modals: opacity+scale 200ms
Fonts
Sans: Geist
Mono: Geist Mono
No serif usage
