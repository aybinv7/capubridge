# Capacitor diagnostics

Status: Hidden

The current Capacitor screens use mock data and are not exposed in production navigation, routing, or the command palette.

Capacitor diagnostics can return only after a dedicated implementation provides real device or project data for plugin inventory, permissions, deep links, configuration, and bridge health. The feature must define its privacy boundary, error states, supported Capacitor versions, and deterministic tests before becoming experimental.

The hidden source is retained temporarily as design reference and is not evidence of shipped capability.
