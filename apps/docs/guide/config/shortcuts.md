## Customizing shortcuts

1. Go to **Settings ‚Üí Shortcuts**
2. Click the shortcut you want to change
3. Press the new key combination
4. Press `Enter` to save

## Reset to defaults

Click **Reset to defaults** to restore all default shortcuts.
