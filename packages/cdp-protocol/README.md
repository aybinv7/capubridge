# CapuBridge CDP protocol package

Typed Chrome DevTools Protocol client and domain adapters used by CapuBridge. The package contains target discovery, transport, DOM, CSS, network, storage, IndexedDB, OPFS, LocalForage, and SQLite bridge contracts.

## Development

- Install dependencies:

```bash
vp install
```

- Run unit tests:

```bash
vp test
```

- Build the package:

```bash
vp pack
```
